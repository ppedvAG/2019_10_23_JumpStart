Bei diesem Beispiel wurde der Effekt rein mit CSS erzielt. Dabei haben sowohl das umschließende Div, als auch das Bild selbst einen :hover-Effekt. Das kann erwünscht sein; allerdings macht es hier auch einen großen Unterschied, ob der Mauszeiger nur über dem Rahmen schwebt oder über dem Bild (und somit zugleich über dem umschließenden Div).
Für diese Lösung wurde eine einfache JavaScript-Funktion verwendet. Bei einem Mouseover über das umschließende Div wird die Transformation für beide Elemente gestartet. Allerdings bleibt das Bild dann auch in dieser Position.
Soll das Bild wieder in die ursprüngliche Position zurückfahren, wenn man nicht mehr mit der Maus darüberschwebt, kann man, wie hier, eine weitere Funktion für onmouseout aufrufen. Eine andere Möglichkeit wäre, die toggle()-Funktion von jQuery zu benutzen.
